# YUMEZZ — Aliff's Personal Website

A beautiful, dark-themed personal portfolio website built with HTML, CSS, and vanilla JavaScript.

## 🐛 Bug Fixed
- **Image path inconsistency**: Changed `image/logo.png` to `images/logo.png` for consistency across all image references

## 📁 Folder Structure

Before uploading to GitHub, organize your files like this:

```
yumezz-website/
├── index.html
├── about.html
├── hobbies.html
├── food.html
├── education.html
├── gallery.html
├── contact.html
├── style.css
├── transitions.css
├── images/
│   ├── logo.png
│   └── me.png
├── video/
│   └── dailylife.mp4
├── music/
│   └── music.mp3
└── README.md (this file)
```

## ✨ Features

- Smooth scroll animations
- Persistent music player across pages
- Responsive design with modern styling
- Dark theme with purple accent color
- Video integration
- Intersection Observer for scroll reveals

## 🎨 Color Scheme

- **Primary**: #9b5de5 (Purple)
- **Dark**: #06060a
- **Text**: #f0eeff

## 📝 What You Need to Add

1. **HTML Pages**: Create `about.html`, `hobbies.html`, etc. using the same structure as `index.html`
2. **External CSS**: Create `style.css` and `transitions.css` files
3. **Media Files**:
   - Logo image: `images/logo.png`
   - Your photo: `images/me.png`
   - Video: `video/dailylife.mp4`
   - Music: `music/music.mp3`

## 🚀 Deploying to GitHub Pages

Once uploaded:
1. Go to **Settings** → **Pages**
2. Select **main** branch as source
3. Your site will be live at: `https://yourusername.github.io/yumezz-website`

## 📌 Notes

- All inline styles in the `<style>` tag are included in `index.html`
- Music auto-plays (users may need to interact with page first due to browser policies)
- Logo has a fallback to text if image fails to load
